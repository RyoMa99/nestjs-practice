# practice Nest JS
## nestjs cliのインストール
```bash
  yarn init -y
  yarn global add @nestjs/cli
```

## nestjs projectの作成
```bash
  nest new sample
```

## nestjs project立ち上げ
```bash
  cd sample
  yarn start:dev
```
